# FiscalFlow-ModaPE
Sistema de gestão fiscal e BI para o polo têxtil de Pernambuco.

## Status do Projeto: Fase 1 (MVP)
Atualmente, o projeto concluiu sua **Fase 1**, focada em:
- Estruturação da lógica de dados e regras de negócio.
- Automação de cálculos fiscais e geração de base de dados (.CSV).
- Criação de protótipos de visualização (BI).

## Próximos Passos (Roadmap)
Este não é apenas um exercício, mas o embrião de um sistema real. Pretendo evoluí-lo para:
1. **Fase 2:** Transformar o código em uma **Interface Web interativa** (usando Streamlit).
2. **Fase 3:** Implementar um **Banco de Dados relacional** (SQL) para persistência segura.
3. **Fase 4:** Criar um módulo de **Previsão de Demanda** com Inteligência Artificial para ajudar na compra de tecidos.
